# Multi-Business Delivery Platform

A comprehensive FastAPI-based platform supporting multiple businesses, each with its own database, within a single application ecosystem.

### Core Architecture
- **Multi-Tenant Database**: Each business has its own PostgreSQL database
- **Global Database**: Shared user, business, and delivery management
- **Role-Based Access**: Customer, Vendor, Delivery Boy, Admin
- **JWT Authentication**: Secure token-based authentication

### User Features
- Registration/Login with phone/email
- Multi-cart system
- Live order tracking
- Wallet & reward points
- Address management
- Real-time notifications
- AI-powered recommendations

### Vendor Features
- Business registration via Access Keys
- Product/Item management
- Order management system
- Business analytics
- Staff management
- Inventory control

### Delivery System
- Delivery boy registration & verification
- Real-time GPS tracking (WebSocket)
- Route optimization
- Earnings dashboard
- Delivery assignment system

### Admin Panel
- Business management
- Access Key generation
- User management
- Commission settings
- Analytics dashboard
- Fraud detection
